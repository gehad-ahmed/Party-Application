# 🎉 Party / Event Landing Page

> 🔗 **Live Demo:** [https://gehad-ahmed.github.io/Party-Application/](https://gehad-ahmed.github.io/Party-Application/)

A responsive event landing page featuring a live countdown timer, animated navigation and interactive sections, built with HTML, CSS and jQuery.

## 📖 Overview

A single-page landing site for an event/party, with a countdown to the event date, a slide-in mobile navigation drawer, smooth-scroll navigation, an expandable details section, and a contact form with a live character counter.

## ✨ Features

- ⏳ **Live countdown timer** to the event (days / hours / minutes / seconds)
- 📱 **Slide-in sidebar navigation** for mobile
- 🖱️ **Smooth-scroll** navigation between sections
- 🔽 **Accordion / slide-toggle** details section
- ✉️ **Contact form** with a remaining-characters counter
- 💯 Fully responsive layout

## 🛠️ Tech Stack

- HTML5 & CSS3
- jQuery
- JavaScript (ES6+)

## 🚀 Getting Started

```bash
git clone https://github.com/gehad-ahmed/Party-Application.git
cd Party-Application
```

Then open `index.html` in your browser (or use the VS Code **Live Server** extension).
